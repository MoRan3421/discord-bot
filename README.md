## 🌟 关于 神殿机器人

神殿机器人 致力于为 Discord 服务器带来无比温馨和友好的氛围。这里的每一个功能，都倾注了开发者的心思，只为让每位用户都能在互动中感受到满满的爱和欢乐。

---

## 🎀 核心功能

- **温馨问候**：自动发送亲切的欢迎信息，让新成员感受家的温暖。  
- **互动小游戏**：内置多个趣味小游戏，时刻带来笑声与乐趣。  
- **音乐陪伴**：智能播放列表，为你打造轻松氛围的专属背景音乐。  
- **频道管理**：提供简单高效的工具，协助维护和谐的聊天环境。
